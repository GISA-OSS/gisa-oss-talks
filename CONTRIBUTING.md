# Contributing your talk

Thank you for sharing your open‑source–related talk!

## What can be added
- Talks with a clear open‑source relation
- Public conferences, meetups, community events

## Rules
- Only final presentations, no work in progress slides
- Slides and videos are optional but encouraged
- If externally hosted slides and/or video, reference the attributed license (if any)

### Slides
- For uploads to this repository:
  - PDF format only (no PPTX)
  - Keep files reasonably small (<20 MB)
  - Must be redistributable under a Creative Commons license (be mindful of stock photos and third-party content)
- Alternatively: External Linking allowed
- Filenames must be descriptive

### Videos
- Do **not** upload videos to this repository
- Provide a public link instead (YouTube, Vimeo, media.ccc.de, …)

### How to add a talk
0. Connect your GitHub profile with [GISA-OSS](https://github.com/GISA-OSS) (contact ctos@gisa.de for assistance)
1. Add the talk to `README.md`
2. Add slides to `slides/<year>/` (if applicable)
3. Create a pull request (from a branch or fork)
